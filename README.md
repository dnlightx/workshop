# TaskRewards - Gamified Productivity App

## Features

- User Authentication
- Task Management
- Pomodoro Timer
- Habit Tracking
- Rewards System
- Progress Analytics
- Premium Features
- Dark/Light Mode

## Tech Stack

- Frontend: React.js
- Backend: Flask
- Database: PostgreSQL
- Authentication: JWT
- State Management: Redux
- UI Framework: Material-UI
- Charts: Chart.js
